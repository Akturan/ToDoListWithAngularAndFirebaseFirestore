# Firebase Firestore and Angular ToDo List App

Navigate to your environment.ts file and paste the config details inside the environment object as firebaseConfig.

```
export const environment = {
    production: true,
    firebaseConfig: {
       apiKey: "",
       authDomain: "",
       databaseURL: "",
       projectId: "",
       storageBucket: "",
       messagingSenderId: ""
     }
};
```
